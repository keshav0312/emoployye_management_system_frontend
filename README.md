# Employee Management System - Frontend (React + Vite)

This is the frontend part of the Employee Management System.  
It provides a user interface to add, view, edit, and delete employees.  
The frontend communicates with the backend using REST API calls.

---

##  Tech Stack
| Technology | Use |
--------------------------
| React (Vite) | UI framework |
| Axios | API calls |
| React Router | Page navigation |
| Bootstrap  | Styling |
